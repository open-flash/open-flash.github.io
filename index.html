Open Flash

Open Flash is a project aiming to create an open-source media player for SWF files to keep Flash games and animations accessible. All the components are reliable, reusable and available as individual libraries, enabling you to use them for your own projects.

Check the player demo.

What is Open Flash

Player

The main goal of the project is to create a player for SWF files. See domu-player.

Libraries

All the components are provided a small reusable and reliable libraries.

Database

Open Flash maintains a registry of public SWF files and a database of test samples.

Documentation

Open Flash centralizes documentation around the SWF file format.
