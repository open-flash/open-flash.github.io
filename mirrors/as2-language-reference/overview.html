Overview of the ActionScript 2.0 Language Reference

The ActionScript Language Reference provides information on the syntax and usage of every officially supported element in the ActionScript language. It includes a section on language elements; alphabetical entries for class elements; a section on deprecated items; and a complete index containing all entries.

This version of the Language Reference includes the following new features to help you better understand the ActionScript language:

  • Inheritance path and list of inherited elements
  • Function and property signatures that indicate the data type of parameters and return values
  • Indication if an element is static

The ActionScript Language Reference is a reference manual for the Flash Player application programming interface (API). It is intended to be used with other instructional media, including, but not limited to, Learning ActionScript in Flash, sample applications installed with the authoring tool, and resources on the Adobe website, such as the Adobe Developer Connection.
